# Transparency: How This Project Was Built

This project was researched, written, and built using [Claude Code](https://claude.ai/code) with Anthropic's Claude Opus 4.6 (1M context) model.

The human-AI collaboration includes:
- Research synthesis across academic papers, sustainability reports, and public records
- Data structure design and schema validation
- Website code (Astro, TypeScript, Tailwind CSS)
- Documentation and methodology writing
- Ongoing dataset maintenance and updates

We disclose this for the same reason we expect AI providers to disclose their water usage: transparency is the point.

## How This Works in Practice

- A human defined the methodology, selected the disclosure dimensions, and reviews every dataset update.
- Every claim links to a published source. The dataset is auditable.
- AI-assisted research is faster but not less rigorous. The same standard applies: if we can't cite it, we don't say it.
- The same methodology is applied uniformly to every provider.
